## Introduction

OnchainKit is a tool designed for building onchain applications. It provides a set of components and utilities to simplify the development process of blockchain-based applications.

## Key Features

- **Wallet Integration**: Connect to various blockchain wallets with ease
- **UI Components**: Pre-built components with shadcn UI styling
- **Network Flexibility**: Support for multiple networks including mainnet and testnet environments
- **React Hooks**: Custom hooks for blockchain data and state management
